# HookDump — version history

The historical releases of HookDump, recovered from the author's archives and published
openly. Most folders hold the period **release binaries** (`HOOKDUMP.EXE`, `HOOKDMP.DLL`,
`HOOKDUMP.INI`, plus `HOOKDMP.TXT` notes or `HOOKDUMP.HLP` help). One early snapshot also
contains **source**.

| Folder | Version | Date (archive) | Contents |
|--------|---------|----------------|----------|
| [`1997-02-early-source/`](1997-02-early-source) | early 1.x | 1997-02-04 | **source** (`HOOKDUMP.PAS`, `HOOKDMP.PAS`, `IOGDI.TPW`) + EXE/DLL |
| [`1.5/`](1.5)   | 1.5  | 1997-02-04 | EXE, DLL, INI |
| [`2.0b/`](2.0b) | 2.0 beta | 1997-02-13 | EXE, DLL, INI, TXT |
| [`2.5b/`](2.5b) | 2.5 beta | 1997-04-27 | EXE, DLL, INI, TXT |
| [`2.6b/`](2.6b) | 2.6 beta | 1997-10-11 | EXE, DLL, INI, TXT |
| [`2.7/`](2.7)   | 2.7  | 1997-10-24 | EXE, DLL, INI, HLP |
| [`2.8/`](2.8)   | 2.8  | 1998-01-09 | EXE, DLL, INI, HLP |

`_hide-tool/` — the small companion `HIDE` utility (`HIDE.PAS` + `HIDE.EXE`).

The current, fully-commented source lives in [`../src/`](../src). The complete recovered
working tree (Delphi port, help sources, backups) is preserved under `_backup/`.

> These are 1990s 16-bit Windows binaries, kept as historical artifacts. They do not run
> on modern 64-bit Windows and must not be used to monitor anyone without consent — see
> [../NOTICE.md](../NOTICE.md).
