# Historical originals — DO NOT MODIFY

These are the **byte-exact original source files**, exactly as recovered from the author's
1996–1998 archives, in their original **Windows-1251 (CP1251)** encoding. Their Russian
comments and string literals therefore render as mojibake when viewed as UTF-8 (for
example, on GitHub) — that is expected and intentional.

They are kept untouched so the unmodified historical source is never lost.

The **working, modern copies** — re-encoded to UTF-8 and given a provenance header (author,
date, links) — live in the sibling folders:

- [`../original-16bit/`](../original-16bit) — the original 16-bit Borland Pascal for Windows
- [`../port-32bit-delphi/`](../port-32bit-delphi) — the 32-bit Delphi port
- [`../successor-hookproj/`](../successor-hookproj) — the small Delphi hook demo

To read the Russian text in these originals: `iconv -f CP1251 -t UTF-8 <file>`.
